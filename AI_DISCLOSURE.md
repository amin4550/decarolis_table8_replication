# Generative AI disclosure

ChatGPT and Gemini were used for coding support, workflow planning, debugging, and documentation. Specifically, it was used to:

- interpret the Decarolis replication ReadMe and the relevant Stata do-files;
- translate the Table 8 Stata workflow into Python;
- debug the construction of the region-year dataset, LIPSA weight concentration, clustered OLS, and clustered 2SLS estimates;
- format regression outputs; and
- draft the replication report and README documentation.

The submitting students are responsible for reviewing every part of the code, confirming that the output is reproducible, verifying the reported results against the original paper, and being able to explain the empirical strategy and all methodological choices.
