# Data instructions

This folder is where the author replication dataset should be placed.

The required file is:

```
master_data_file.dta
```

It must be placed at:

```
data/master_data_file.dta
```

The dataset is not included in this GitHub repository or submission ZIP file. To obtain it, download the replication package from openICPSR:

```
https://www.openicpsr.org/openicpsr/project/112933/version/V1/view
```

After downloading and extracting the ZIP file, locate `master_data_file.dta` and copy it into the `data/` folder.

The original replication materials describe `master_data_file.dta` as the dataset used for the analysis. The dataset can be reconstructed from public CMS, Q1Medicare, formulary, and IPUMS sources, but this Python replication uses the author-provided `master_data_file.dta`.
